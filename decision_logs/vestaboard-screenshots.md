# Vestaboard Screenshots

- For screenshot updates, always ask the user for the board-specific API key unless the key has already been provided for that board type in the current task.
- Always ask the user for the board-specific Vestaboard web URL, for example `https://web.vestaboard.com/board/<board-id>/messages`; do not assume one board URL works for another board.
- Flagship screenshots must retain the thick physical frame around the board. Do not crop Flagship screenshots down to only the tile grid.
